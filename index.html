CONOCIMIENTOS

HTML "Lenguaje de Marcado de Hipertexto" es el estándar utilizado para la creación y diseño de páginas web. Es un lenguaje de marcado que estructura el contenido de una página web mediante el uso de elementos y etiquetas.

CSS "Hojas de Estilo en Cascada" es un lenguaje de diseño que se utiliza para controlar el estilo y la apariencia visual de las páginas web.

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Se utiliza para agregar interactividad a las páginas web.

Las metodologías ágiles son un conjunto de enfoques y prácticas para el desarrollo de software. Se centran en responder de manera ágil a los cambios en los requisitos del proyecto y en satisfacer las necesidades del cliente de manera rápida y efectiva.
